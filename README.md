# Insurance-Claims-Analysis
# OVERVIEW
This project focuses on analysing health insurance claims data to uncover trends in claim behaviour, cost drivers, and policy utilization. The workflow includes data cleaning using Excel and the development of an interactive Power BI dashboard to generate actionable insights for decision-making.

# OBJECTIVES
Clean and transform raw insurance claims data
Ensure data accuracy and consistency for analysis
Identify key trends in claims and costs
Build an interactive dashboard for visual insights

# Tools & Technologies
Microsoft Excel (Data Preparation & Cleaning)
Power BI (Data Visualization & Dashboarding)
DAX (Calculated measures & KPIs)
